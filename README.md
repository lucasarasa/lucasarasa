Tenho foco no desenvolvimento de aplicações web completas, criando e consumindo APIs, integrando sistemas e trabalhando com bancos de dados relacionais.

Já atuei em projetos reais desempenhando funções como Desenvolvedor Full Stack, Front-end, com experiência em cloud (AWS), práticas de DevSecOps e também atuando com QA. Sou apaixonado por tecnologia, aprendizado contínuo e boas práticas de desenvolvimento, buscando sempre escrever código limpo, organizado e escalável.
